Preguntas frecuentes

Todo claro antes de empezar.

¿Necesito saber programar?

No para crear plantillas. El editor es visual. Para automatizarlas puedes usar n8n o Make sin código, o integrar la API desde tu aplicación.

¿Puedo probarlo sin tarjeta?

Sí. Tu cuenta nueva recibe 50 créditos de bienvenida y permite crear hasta 3 plantillas.

¿Las imágenes salen en alta calidad?

Sí. Los renders y las exportaciones están preparados para entregar imágenes de alta resolución.

¿Qué pasa cuando se terminan mis créditos?

Los renders se detienen para evitar cargos inesperados. Puedes activar el plan que corresponda a tu volumen mensual.

¿Funciona con n8n?

Sí. Solo necesitas un nodo HTTP Request, tu API key, el ID de la plantilla y el JSON con las capas que deseas cambiar.
